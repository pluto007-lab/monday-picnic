# Monday Picnic

![トップページ](img/preview.png)

月曜日が待ち遠しくなる、韓国風セレクト雑貨店のランディングページです。

## 使用技術

- HTML5
- CSS3
- JavaScript

## ディレクトリ構成

```
Monday-Picnic/
├── index.html
└── img/
```

## コンセプト

「憂鬱な月曜日の朝に、小さな魔法を。」

韓国インテリアや雑貨の世界観をイメージし、
淡い色合いと大きな写真を取り入れた架空のセレクトショップLPを制作しました。

## 主な内容

- ヒーロービジュアル
- 商品カテゴリ紹介
- ブランドコンセプト
- メール登録フォーム
- レスポンシブ対応

## 工夫した点

- CSSアニメーションによる自然な表示演出
- 余白を活かした読みやすいレイアウト
- 柔らかい配色でブランドイメージを表現
- シンプルな構成で視線誘導を意識したデザイン
